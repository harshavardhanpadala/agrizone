# AgriZone
A Mandal-Level Agricultural Marketplace designed to connect farmers with necessary equipment, labor, and infrastructure.

## Tech Stack
* **Frontend:** React, Tailwind CSS
* **Backend/Database:** Supabase
* **Deployment:** Vercel

## Features
* Mandal-level location filtering.
* Marketplace for Equipment, Labor, and Infrastructure.
* User-friendly dashboard for managing and viewing listings.

## How to Run
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your `.env` file with your Supabase credentials.
4. Run locally: `npm run dev`.
